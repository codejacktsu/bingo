# Sound Effects

## Required Files

Place your sound effect files in this directory:

- `draw.mp3` - Sound played when a number is drawn

## Where to Get Sounds

### Free Sound Resources:
1. **Freesound.org** - https://freesound.org/
   - Search for: "bell", "ding", "chime", "pop"
   - Download as MP3 format

2. **Zapsplat** - https://www.zapsplat.com/
   - Free sound effects (with attribution)
   - Search for notification sounds

3. **Create Your Own**:
   - Use online tone generators
   - Record simple sounds
   - Use Audacity (free audio editor)

## Recommended Sound Characteristics

- **Format**: MP3 (best browser compatibility)
- **Duration**: 0.5 - 1 second (short and crisp)
- **Volume**: Normalized to avoid clipping
- **File Size**: < 50KB
- **Type**: Simple bell, chime, or notification sound

## Fallback

If no sound file is provided, the application will still work without audio.
The AudioPlayer component gracefully handles missing audio files.

## Example Sounds

Good sound types for bingo number drawing:
- Bell chime
- Soft ding
- Pop sound
- Notification beep
- Glass clink

Avoid:
- Long sounds (> 2 seconds)
- Voices or speech
- Complex musical sounds
- Harsh or jarring sounds
